GPU PARALLEL FLOATING-POINT SIMULATOR
Computer Organization and Architecture Project
=====================================================

PROJECT OVERVIEW
----------------
Educational web application demonstrating all 5 Course Outcomes (CO1-CO5)
through an interactive GPU parallel computing simulator.

COURSE OUTCOMES COVERED
-----------------------
✓ CO1: Instruction Cycle (Fetch-Decode-Execute)
✓ CO2: Performance Evaluation (Speedup, Efficiency, Throughput)
✓ CO3: Interrupt Handling (Context Switching, ISR)
✓ CO4: I/O Interfacing (Input/Output Operations)
✓ CO5: GPU Architecture (SIMD, Thread Blocks, Parallelism)

TECHNOLOGY STACK
----------------
Backend:  Python, FastAPI, NumPy, Multiprocessing
Frontend: HTML5, CSS3, JavaScript, Chart.js
Server:   Uvicorn (ASGI)

SETUP INSTRUCTIONS
------------------
1. Install dependencies: pip install -r requirements.txt
2. Run server: python run_server.py
3. Open browser: http://localhost:8000

For detailed setup instructions, see SETUP.md

KEY FEATURES
------------
• Interactive web interface with real-time visualization
• Automatic CO activation during simulation workflow
• Performance comparison charts (CPU vs GPU)
• Thread block visualization
• Instruction cycle step-by-step execution
• Interrupt handling with context switching
• PDF export functionality
• Console logging with labeled CO messages

DEMONSTRATION FLOW
------------------
1. Initialize Dataset → CO4 activates (I/O operations)
2. Run CPU Simulation → CO1 activates (instruction cycle)
3. Run GPU Simulation → CO5, CO3, CO2 activate automatically
4. View performance charts and metrics
5. Export results as PDF
